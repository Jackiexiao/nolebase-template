> [!NOTE] 
> 	方法的形参是可以变化的
> 	格式：属性类型...名字
> 	int...args
> 	底层：
> 		可变参数的底层是一个数组
> 		只不过不需要我们自己创建了，Java会帮我们创建好
> 	小细节：
> 		1、再方法的形参中最多只能写一个可变参数
> 		2、方法中，如果除了可变参数以外，还有其他形参，那么可变参数要写在最后面
```Java
public class Test {  
    public static void main(String[] args) {  
        System.out.println(getSum(1, 2, 3, 4, 5));  
    }  
  
    public static int getSum(int... nums) {  
        int sum = 0;  
        for (int num : nums) {  
            sum += num;  
        }  
        return sum;  
    }  
}
```
