# 变量加static和不加static的区别

![](../../img/Pasted%20image%2020251129162427.png)

### 4. 什么时候该用 static？

1. **共享数据：** 当所有对象都需要共享同一个值时（例如：`Math.PI`，或者全班的班费总额）。
2. **常量：** 通常配合 `final` 一起使用，定义全局常量（例如：`public static final String DB_URL = "..."`）。
3. **方便调用：** 像工具类的方法或变量，不需要创建对象就能直接用（例如：`System.out` 中的 `out` 就是个静态变量）。

### 总结

- **不加 static** = **私有笔记**（人手一本，你写你的，我写我的）。
- **加 static** = **教室黑板**（全班一本，谁上去改了一个字，所有人看到的都变了）。
