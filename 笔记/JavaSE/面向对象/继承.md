## super 可以调用父类里的属性
### 语法
```java
	super(参数列表); // 必须位于子类构造方法的第一行
```
### 示例
```Java
	class Animal {
    private String name;
    
    // 父类构造方法
    public Animal(String name) {
        this.name = name;
        System.out.println("Animal构造方法被调用，name: " + name);
    }
}

class Dog extends Animal {
    private String breed;
    
    // 子类构造方法
    public Dog(String name, String breed) {
        super(name); // 调用父类构造方法，必须放在第一行
        this.breed = breed;
        System.out.println("Dog构造方法被调用，breed: " + breed);
    }
}

public class Test {
    public static void main(String[] args) {
        Dog myDog = new Dog("Buddy", "Golden Retriever");
    }
}
```
### 输出
```txt
	Animal构造方法被调用，name: Buddy
	Dog构造方法被调用，breed: Golden Retriever
```
