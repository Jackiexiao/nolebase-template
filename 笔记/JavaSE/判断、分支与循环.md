### default 的位置和省略
	default可以省略，语法不会有问题，但是不建议省略。
	default不一定是写在最下面的，我们可以写在任意位置，只不过习惯写在最下面

### case穿透
	语句体中没有没有写break导致的
	执行流程：
		如果匹配上了，就会执行对应的语句体，如果发现break，那么就结束整个switch语句。
		如果没有发现break，那么程序就会继续执行下一个语句体，一直遇到break或者右大括号为止
	使用场景：
		如果多个case语句体重复了，那么我们考虑用case穿透去简化代码

### switch 新特性
	jdk12 以后推出的新特性
```java
	case "北京拉面" -> {  
	    System.out.println("吃北京拉面");  
	}  
	case "上海拉面" -> {  
	    System.out.println("吃上海拉面");  
	}  
	default -> {  
	    System.out.println("吃其他拉面");  
	}
```
	如果大括号里面只有一行代码，大括号可以省略
```java
	case "北京拉面" -> System.out.println("吃北京拉面");  
	case "上海拉面" -> System.out.println("吃上海拉面");  
	default -> System.out.println("吃其他拉面");
```
