## 自己处理（捕获异常）

![Pasted image 20251031172627](../../img/Pasted%20image%2020251031172627.png)

> 目的：当代码出现异常时，可以让程序继续向下执行。

```Java
int[] arr = {1,2,3,5};
try{
    System.out.println(arr[10]);
}catch (ArrayIndexOutOfBoundsException e){
    System.out.println("索引越界了");
}

System.out.println("test");
```
