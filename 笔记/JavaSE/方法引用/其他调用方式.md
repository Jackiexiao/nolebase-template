## 使用类名引用成员方法

![[Pasted image 20251031104717.png]]

```Java
ArrayList<String> list = new ArrayList<>();
Collections.addAll(list,"aaa","bbb","ccc");
list.stream().map(String::toUpperCase).forEach(System.out::println);
```

## 抽象方法形参的详解：

<!-- prettier-ignore -->
> [!NOTE] Title
> 	第一个参数：
> 		表示引用方法的调用者，决定了可以引用那些类中的方法
> 		在Stream流当中，第一个参数一般都表示流里面的每一个数据
> 		假设流里面的数据是字符串，那么使用这种方式进行方法引用，只能引用String这个类中的方法
> 	第二个参数到最后一个参数：
> 		跟被引用方法的形参保持一致，如果没有第二个参数，说明被引用的方法需要时无参的成员方法
> 	局限性：
> 		不能引用所有类中的成员方法
> 		是跟抽象方法的第一个参数有关，这个参数是什么类型的，那么就只能引用这个类中的方法

```Java
ArrayList<Student> list = new ArrayList<>();
Collections.addAll(list, new Student("张三", 20),
        new Student("李四", 30),
        new Student("王五", 40));
String[] array = list.stream().map(Student::getName).toArray(String[]::new);
```

## 引用数组的构造方法

![[Pasted image 20251031110600.png]]

```Java
ArrayList<Integer> list = new ArrayList<>();
Collections.addAll(list,1,2,3,4,5,6,7,8,9);
//数组中类型必须和流中元素类型一致
Integer[] array = list.stream().toArray(Integer[]::new);
System.out.println(Arrays.toString(array));
```
