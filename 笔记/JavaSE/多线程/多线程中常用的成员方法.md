> [!NOTE] 
> 	1、如果我们没有给线程设置名字，线程有默认的名字（格式为：Thread-X X表示序号，默认从哦开始）
> 	2、我们给线程设置名字，可以使用set方法，也可以使用构造函数
> 		注意：使用构造函数去给线程设置名字，要要在继承Thread中子类中实现构造方法，不然代码会报错

```java
public class MultiDemo04 {
    public static void main(String[] args) {
        MyThread02 t1 = new MyThread02("线程1");
        MyThread02 t2 = new MyThread02("线程2");
        t1.start();
        t2.start();
    }
}

public class MyThread02 extends Thread{
    public MyThread02() {
    }

    public MyThread02(String name) {
        super(name);
    }

    @Override
    public void run() {
        for (int i = 0; i < 100; i++) {
            System.out.println(getName() + "运行，i=" + i);
        }
    }
}
```

<!-- prettier-ignore -->
> [!NOTE] 
> 	当JVM虚拟机启动后会自动启动多条线程，其中有一条就是main线程（就是去调用main方法，并执行方法里面的代码，之前写的代码都是运行在main线程当中的）

> [!NOTE] sleep方法
> 	1、那条线程执行到这个方法，就会在这里停留对应的时间
> 	2、方法的参数：就表示睡眠的时间，单位毫秒
> 	3、当时间到了之后，会继续的执行下面的代码
