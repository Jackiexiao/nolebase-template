# 第一种（没有上限的）

```Java
public class MultiDemo20 {
    public static void main(String[] args) throws InterruptedException {
        MyThread20 t1 = new MyThread20();
        MyThread20 t2 = new MyThread20();
        MyThread20 t3 = new MyThread20();
        ExecutorService pool = Executors.newCachedThreadPool();
        pool.submit(t1);
        Thread.sleep(1000);
        pool.submit(t2);
        Thread.sleep(1000);
        pool.submit(t3);
        pool.shutdown();
    }
}

public class MyThread20 implements Runnable{
    @Override
    public void run() {
        System.out.println(Thread.currentThread().getName() + "线程开始运行");
    }
}
```

# 第二种

```Java
public class MultiDemo20 {
    public static void main(String[] args) throws InterruptedException {
        MyThread20 t1 = new MyThread20();
//        ExecutorService pool = Executors.newCachedThreadPool();
		//参数代表线程池中线程的数量
        ExecutorService pool = Executors.newFixedThreadPool(3);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.submit(t1);
        pool.shutdown();
    }
}

public class MyThread20 implements Runnable{
    @Override
    public void run() {
        System.out.println(Thread.currentThread().getName() + "线程开始运行");
    }
}
```
