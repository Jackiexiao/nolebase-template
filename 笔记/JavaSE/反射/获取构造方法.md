# 有泛型和没泛型的区别

| 特性                 | 没泛型 (Raw Type)      | 有泛型 (Generic Type)  |
| :------------------- | :--------------------- | :--------------------- |
| Class 类型           | Class                  | Class<Student>         |
| Constructor 类型     | Constructor            | Constructor<Student>   |
| newInstance() 返回值 | Object                 | Student                |
| 类型转换             | 需要手动强转 (Student) | 自动对应，无需强转     |
| 安全性               | 运行时才发现错误       | 编译时就能发现部分错误 |
| IDE 警告             | 有黄色警告             | 无警告（代码更干净）   |

```Java
public class MyReflect02 {
    public static void main(String[] args) throws ClassNotFoundException, NoSuchMethodException, InvocationTargetException, InstantiationException, IllegalAccessException {
        Class<?> aClass = Class.forName("javaStudy.myreflect.Student");
        //获取所有公共的构造方法
        Constructor<?>[] cons = aClass.getConstructors();
        for (Constructor<?> con : cons) {
            System.out.println(con);
        }

        System.out.println("=============================");

        //获取所有的构造方法，包括私有的和受保护的
        Constructor<?>[] decons = aClass.getDeclaredConstructors();
        for (Constructor<?> decon : decons) {
            System.out.println(decon);
        }

        System.out.println("==============================");
        //获取单个构造方法
        Constructor<?> con1 = aClass.getDeclaredConstructor();
        System.out.println(con1);
        //获取单个有参构造
        Constructor<?> con2 = aClass.getDeclaredConstructor(String.class);
        System.out.println(con2);

        Constructor<?> con3 = aClass.getDeclaredConstructor(String.class, int.class);
        //私有方法要通过这种方法创建对象，需要临时取消权限校验
        con3.setAccessible(true);
        Student stu = (Student) con3.newInstance("张三", 23);
        System.out.println(stu);
    }
}
```

# Student对象

```java
public class Student {
    private String name;
    private int age;

    public Student() {
    }

    public Student(String name){
        this.name = name;
    }

    protected Student(int age) {
        this.age = age;
    }

    private Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```
