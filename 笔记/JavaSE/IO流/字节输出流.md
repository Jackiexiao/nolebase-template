字节输出流

<!-- prettier-ignore -->
> [!NOTE]
> 	创建字节输出流对象
> 		细节1：
> 			参数是字符串表示的路径或者是File对象都是可以的
> 		细节2：
> 			如果文件不存在会自动创建一个文件，但是父级路径必须是正确的
> 		细节3：
> 			如果文件已经存在，会清空原有的文件再写入
> 	写数据
> 		write的参数是整数，但是写到文件里的是ASCII表中对应的字符
> 	释放资源：
> 		每次使用完流都需要释放资源（解除资源的占用）

```JAVA
FileOutputStream fos = new FileOutputStream("src\\javaStudy\\MyIO\\a.txt");
fos.write(97);
fos.close();
```

## FileOutStream写数据的三种方式

![[Pasted image 20251103150952.png]]

<!-- prettier-ignore -->
> [!NOTE] Title
> 	参数len表示的是写入个数，不是结束索引

```Java
FileOutputStream fos = new FileOutputStream("src\\javaStudy\\MyIO\\a.txt");
byte[] bytes = {97, 98, 99};
fos.write(bytes);
fos.close();

FileOutputStream fos = new FileOutputStream("src\\javaStudy\\MyIO\\a.txt");
byte[] bytes = {97, 98, 99, 100, 101};
fos.write(bytes,1,3);
fos.close();

FileOutputStream fos = new FileOutputStream("src\\javaStudy\\MyIO\\a.txt");
String str = "fengling";
byte[] bytes = str.getBytes();
fos.write(bytes);
fos.close();
```

## FileOutStream写数据的俩个问题

![[Pasted image 20251103151707.png]]

<!-- prettier-ignore -->
> [!NOTE] 换行写
> 	再次写出一个换行符就行
> 	Windows：\r\n
> 	Linux：\n
> 	Mac：\r
> 	细节：
> 		在Windows中，Java进行了优化，只需要输入一个\r或者\n在底层Java会帮我们自动补全

```Java
FileOutputStream fos = new FileOutputStream("src\\javaStudy\\MyIO\\a.txt");
String str = "fengling\r\nhhh";
byte[] bytes = str.getBytes();
fos.write(bytes);
fos.close();
```

> [!NOTE] 续写
> 	如果要续写，打开续写开关就可以
> 	开关位置：创建对象的第二个参数，默认为false
> 	手动传递true，此时创建文件就不会清空
