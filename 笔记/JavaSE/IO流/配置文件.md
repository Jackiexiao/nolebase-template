## Properties和IO流结合的操作

### 保存到配置文件

```Java
public class IODemo40 {
    public static void main(String[] args) throws IOException {
        Properties prop = new Properties();
        prop.put("username","root");
        prop.put("password","123456");
        FileOutputStream fos = new FileOutputStream("src/javaStudy/MyIO/prop.properties");
        //保存数据,参数一：输出流，参数二：注释
        prop.store(fos,"test");
        fos.close();
    }
}
```

### 加载配置文件

```Java
public class IODemo41 {
    public static void main(String[] args) throws IOException {
        Properties prop = new Properties();
        FileInputStream fis = new FileInputStream("src/javaStudy/MyIO/prop.properties");
        prop.load(fis);
        fis.close();
        System.out.println(prop.getProperty("username"));
        System.out.println(prop.getProperty("password"));
    }
}
```
