## 阶段三：专项强化（贯穿全程）

### 1. 算法练习（**立即开始** - 已学完JavaSE集合部分）
- **平台**：LeetCode + 牛客网
- **题单**：
  - Hot100（[链接](https://leetcode.cn/problem-list/QQcuyKam/)）
  - LeetCode 75
  - 面试经典150题
  - 高频SQL50题
- **方法**：
  - 每日1-2题，按分类刷（简单→中等→困难）
  - 5分钟无思路直接看题解
  - 反复刷至能默写（Hot100建议刷3遍以上）
- **资源**：[灵茶山艾府算法精讲](https://space.bilibili.com/206214/lists/842776?type=season)

### 2. 八股准备（每日整理）
- **核心资源**：
  - [小林coding](https://xiaolincoding.com/)
  - [JavaGuide](https://javaguide.cn/)
  - [二哥](https://javabetter.cn/)
- **方法**：
  - 每学完一个模块，整理该模块八股
  - 结合深度思考记录（非死记硬背）
- **必读书籍**：
  - 《深入理解Java虚拟机》
  - 《MySQL技术内幕》
  - 《Java并发编程的艺术》
  - 《亿级流量系统架构设计与实战》

### 3. 项目实践（进阶阶段后启动）
- **原则**：
  - 至少2个项目（有实习可减至1个）
  - 避免烂大街项目（推荐Java+AI方向）
  - 重点体现个人思考与创新点
- **示例**：
  - 基于开源项目改造（如黑马点评）
  - 添加AI功能模块（如智能推荐、NLP处理）
  - 优化高并发场景（缓存、消息队列、分库分表）
