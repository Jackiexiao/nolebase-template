## 监听鼠标点击和空格键事件
```Java
	//创建按钮对象
	Button jbt = new JButton("点击");  
	jbt.setBounds(0,0,100,50);  
	//监听鼠标点击和空格键事件  
	jbt.addActionListener(new AbstractAction() {  
	    @Override  
	    public void actionPerformed(ActionEvent e) {  
	        System.out.println("点击了按钮");  
	    }  
	});  
	  
	jFrame.getContentPane().add(jbt);
```
