## splice方法

方法可以删除、替换或添加数组元素，它会直接修改原数组

```js
// 删除指定位置的元素
array.splice(startIndex, deleteCount)

// 你的代码示例：
this.list.splice(index, 1) // 从index位置开始，删除1个元素

// 删除数组中的对象
let users = [
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
    { id: 3, name: 'Charlie' },
]
const idToDelete = 2
// 1. 找到要删除对象的索引
const indexToDelete = users.findIndex((user) => user.id === idToDelete)
// 2. 如果找到了，就使用 splice 删除
if (indexToDelete !== -1) {
    users.splice(indexToDelete, 1) // 从 indexToDelete 位置开始，删除 1 个元素
}
console.log(users)
// 输出 (原数组被修改): [{ id: 1, name: 'Alice' }, { id: 3, name: 'Charlie' }]
```

## filter方法

如果你不想修改原数组，而是创建一个不包含特定元素的新数组

```js
// 删除特定值的元素
const newArray = originalArray.filter((item) => item !== valueToDelete)

// 删除特定索引的元素
const newArray = originalArray.filter((item, index) => index !== indexToDelete)

// 删除数组中的对象
let users = [
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
    { id: 3, name: 'Charlie' },
]
const idToDelete = 2
// 使用 filter 创建一个不包含 id 为 2 的对象的新数组
const newUsers = users.filter((user) => user.id !== idToDelete)
console.log(newUsers)
// 输出: [{ id: 1, name: 'Alice' }, { id: 3, name: 'Charlie' }]
console.log(users)
// 输出 (原数组不变): [{ id: 1, name: 'Alice' }, { id: 2, name: 'Bob' }, { id: 3, name: 'Charlie' }]
```


## shift() 方法（删除第一个元素）

