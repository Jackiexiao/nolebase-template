# 全局注册

```js
//全局注册（在main.js中）
//focus是名字，使用时直接在元素上写v-focus
Vue.directive('focus', {
    //inserted是钩子函数，元素插入到DOM中时调用
    inserted(el) {
        console.log(el)
    },
})
```

# 局部注册

```js
//局部注册
  directives: {
    focus: {
      inserted(el) {
        el.focus()
      }
    }
  }
```
