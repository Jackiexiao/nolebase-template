## 1、将按钮类型设为 button 而不是 submit

```html
<button @click="delTask(item)" type="button">删除</button>
```

## 2、使用事件修饰符`.prevent`

```html
<form @submit.prevent>
    <!-- 表单内容 -->
</form>
```

## 3、使用事件对象

> 在处理方法中调用 event.preventDefault()

```html
<form @submit="handleSubmit">
    <!-- 表单内容 -->
</form>

methods: { handleSubmit(event) { event.preventDefault(); // 你的处理逻辑 } }
```

## 4、使用 @click 修饰符

```html
<button @click.prevent="addTask">添加任务</button>
```
