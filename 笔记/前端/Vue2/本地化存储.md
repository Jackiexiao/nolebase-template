```html
<script>
    data(){
    	return{
    		todos: JSON.parse(localStorage.getItem('todos')) || [
    			{id:1,title:'测试1'},
    			{id:2,title:'测试2'}
    		]
    	}
    },
    watch:{
    	todos:{
    		deep:true,
    		handler(newValue){
    			localStorage.setItem('todos',JSON.stringify(newValue))
    		}
    	}
    }
</script>
```
## 要开启深度监听
