## v-for
```ts
	<li v-for="game in games">
      <span class="game-id">{{ game.id }}</span>
      <br />
      <span class="game-name">{{ game.name }}</span>
    </li>
    <li v-for="n in 10">
      <span>{{ n }}</span>
    </li>
```
