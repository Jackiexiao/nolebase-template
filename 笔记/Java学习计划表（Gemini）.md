## 阶段三：专项强化（贯穿全程）

**(此阶段内容和方法不变，请按照自己的节奏持续进行)**

### 1. 算法练习
- **启动时机**：现在就可以开始了！因为 JavaSE 集合部分的基础你已经掌握。
- **方法**：保持 **每日1-2题** 的节奏，周末可回顾。不求快，但求坚持。
- **平台与题单**：同原计划（LeetCode Hot100 是重中之重）。

### 2. 八股准备
- **方法**：每学完一个模块，就去 JavaGuide 或 小林coding 上看对应的面试题，**不是背，是理解**。把理解后的内容用自己的话整理成笔记。

### 3. 项目实践
- **启动时机**：在你做完 Redis 阶段的 “黑马点评” 项目后，就可以开始构思和启动你的第二个项目了。
